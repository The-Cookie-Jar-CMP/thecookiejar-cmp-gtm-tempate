# The Cookie Jar CMP template for Google Tag Manager
This template will get you up and running with The Cookie Jar though using Google Tag Manager.  The Cookie Jar is a Consent Management Provider (CMP) designed to keep the day-to-day management of your websites cookie consent as simple as possible.

To get started you will need to create an account with us on https://cookieconsentjar.com/.  For more information on how to implement The Cookie Jar through Google Tag Manager you can follow this guide: https://cookieconsentjar.com/docs/get-started/quickstart-with-google-tag-manager/
